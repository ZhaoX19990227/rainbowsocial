# 熊猴互动暧昧动作系统设计规格

## 视觉方向

- 调性: 明亮、清爽、带紫色霓虹和蓝色高光，粉色只作为情绪点缀。
- 角色: 小熊更厚实温柔，小猴更灵动俏皮，统一为 Q 版、圆润、头大身小、四肢短圆。
- 气质: flirt / playful / intimate / warm，不走低幼卡通，也不走脏黑夜店风。
- 场景: 移动端聊天动作面板、聊天消息卡片、全屏回放三种核心使用场景。

## 界面输出

### 1. 动作选择面板

- 位置: 聊天输入框上方半屏弹出。
- 分类: 坏一点、再靠近、软绵绵、撩一下。
- 卡片结构: 封面插画 + 动作名 + 一句预览文案。
- 视觉重点: 紫色主调、蓝色高光、玻璃面板和轻颗粒背景。

### 2. 动作卡片与全屏回放

- 聊天气泡卡片: 适合在消息流中展示，突出封面帧和一句预览文案。
- 全屏回放: 以舞台感和柔和发光为主，不做游戏结算页式厚重黑幕。
- 背景层次: 柔光、玻璃感、轻颗粒、呼吸辉光。

### 3. 六个核心动作

- 拍拍屁股
- 摸摸头
- 勾勾手指
- 拽住衣角
- 忽然靠近
- 偷看被抓包

## 动效制作建议

- 角色主动画: `Rive`
- 氛围特效: `Lottie`
- 高速点缀: `sprite sheet`
- 文档/评审演示: 可额外导出 GIF，但不作为正式交付格式

## 可拆分特效层

- 心跳光点
- 星点
- 速度线
- 腮红
- 呼吸辉光
